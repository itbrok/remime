# Remime

**Remime** is a beautiful, minimalist Chrome popup extension designed to keep you focused and mindful throughout your day. 

With a single click, Remime provides a calming interface featuring a real-time 12-hour clock, a time-appropriate greeting, and a persistent "Today's Focus" to-do list. It's the perfect lightweight companion to remind you of your core tasks without the clutter of a full new tab replacement.

## Features

- **🕰️ Minimalist Clock:** A large, easily readable 12-hour clock to keep you anchored in the present.
- **👋 Dynamic Greetings:** Friendly greetings that change automatically based on the time of day (Morning, Afternoon, Evening).
- **✅ Focus To-Do List:** A simple, elegant task manager where you can add, check off, and delete your main goals for the day.
- **💾 Persistent Storage:** Your tasks are saved using Chrome's local storage, ensuring your focus items are always waiting for you when you open the popup.
- **🖼️ Beautiful UI:** A calming background and frosted-glass container styling that is easy on the eyes.

## Installation

Currently, Remime is not available on the Chrome Web Store, but you can easily install it locally as an unpacked extension.

1. **Download the project:** Clone this repository or download the ZIP file and extract it.
   ```bash
   git clone https://github.com/itbrok/remime.git
   ```
2. **Open Chrome Extensions:** Open Google Chrome and navigate to `chrome://extensions/` in your address bar.
3. **Enable Developer Mode:** In the top right corner of the Extensions page, toggle the **"Developer mode"** switch to ON.
4. **Load Unpacked:** Click the **"Load unpacked"** button that appears in the top left.
5. **Select Folder:** Select the folder where you extracted the Remime files.
6. **Pin the Extension:** For easy access, click the puzzle piece icon in your Chrome toolbar and "pin" the Remime extension. 

## Usage

Simply click the Remime icon in your Chrome toolbar!
- Type into the input field and press Enter to add a new focus task.
- Click the checkbox to mark a task as completed.
- Hover over a task and click the red "x" to delete it.

## Permissions

Remime requires the following permission:
- `storage`: Used solely to save your "Today's Focus" list locally on your machine. We do not track or send your data anywhere.
